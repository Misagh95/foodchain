## 📊 Overview

FoodChain provides complete food supply chain transparency from farm to plate.

### Key Features
- 🍎 20-second block time
- 📱 QR-based food tracking
- ⚠️ Allergen alerts
- 🌡️ Freshness tracking
- 📘 Built with TypeScript
